# WeChat OCR Listener

一个基于 **Python + PaddleOCR** 的本地微信消息监听与识别项目。  
通过对微信窗口进行截图、裁剪、OCR 识别，实现对聊天消息的自动提取，为后续的 **AI 自动回复、消息分析、关键词监控、聊天记录处理** 等功能提供基础能力。

---

## 项目简介

本项目的核心目标是：

> **在不直接调用微信内部接口的前提下，通过视觉识别方式读取微信聊天窗口中的消息内容。**

相比依赖注入、Hook 或逆向方案，这种方法：

- 更容易上手
- 对 Python 初学者更友好
- 更适合作为自动化与 AI 助手项目的基础模块
- 可拓展到其他聊天软件或任意文本界面识别场景

项目当前主要基于以下技术：

- **Python**
- **PaddleOCR**
- **PyAutoGUI / Pillow / MSS（截图）**
- **OpenCV（图像预处理）**
- **窗口定位与区域裁剪**
- （可选）**LLM API 接入，用于 AI 回复与分析**

## 项目思路

整个项目的工作流程如下：

```text
微信窗口定位
    ↓
截取聊天区域图像
    ↓
图像预处理（可选）
    ↓
PaddleOCR 文字识别
    ↓
文本清洗 / 去重 / 结构化
    ↓
输出消息内容
    ↓
（可选）发送给 AI 进行分析 / 回复
```
# 模块介绍
- chatbot:用于调用ai回复
- get_words_coordinate:返回屏幕上识别到文字在屏幕上的相对坐标
- get_wx_massage:生成器，用于反复截图识别文字，返回识别到的文字列表
- main_program:用于整合以上三个模块，利用for不停调用生成器监听屏幕消息

# 注意！！！
由于无法自动识别聊天窗口的精确坐标所以需要用户自己在get_wx_massage中设置
